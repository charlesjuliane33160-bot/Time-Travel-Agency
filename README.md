# Time-Travel-Agency
Webapp interactive pour l'exercice TimeTravel Agency


Stack Technique
- **Frontend** : React + Tailwind CSS  
- **Animations / Interactions** : Framer Motion  
- **Chatbot IA** : Lovable.dev (version gratuite, réponses prédéfinies)  
- **Hébergement** : Vercel  

Features
- Page d’accueil avec Hero section vidéo / animation  
- Présentation de l’agence  
- Galerie de 3 destinations temporelles (Paris 1889, Florence 1504, Crétacé -65M)  
- Cards interactives pour chaque époque  
- Informations détaillées par destination  
- Chatbot IA conversationnel (réponses prédéfinies pour l’exo)  
- Formulaire de réservation (destination + dates, validation simple)  

  Outils et IA utilisées
- Code / génération site : Lovable.dev + Framer pour la maquette  
- Chatbot : réponses pré-définies Lovable (simulation IA)  
- Visuels : Midjourney + Runway pour images / assets  
- Autres outils : Bolt.new pour premiers prototypes
